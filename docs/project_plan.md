# 小鲨鱼德州科普小视频统筹方案

## 定位

用“小鲨鱼”这个可爱角色，把德州扑克入门知识讲得轻松、清楚、不吓人。第一条视频建议讲“德州扑克一局到底在比什么”，避免一上来塞太多术语。

## 目标观众

- 对德州扑克感兴趣但完全没入门的人
- 想快速看懂牌桌流程的人
- 喜欢轻松动画/可爱角色讲解风格的人

## 第一条视频主题

标题候选：

- 小鲨鱼 60 秒看懂德州扑克
- 德州扑克到底怎么玩？小鲨鱼带你入门
- 不懂德州？先记住这 3 件事

核心信息：

- 每个人有 2 张手牌
- 桌面最多出现 5 张公共牌
- 用自己的手牌和公共牌凑出最强 5 张牌
- 下注不是乱加码，而是在围绕胜率、位置和信息做决定

## 风格

- 画面：明亮、海洋感、可爱但不要幼稚
- 节奏：每 3-5 秒一个视觉变化
- 语言：中文口语化，少术语，术语出现后立刻解释
- 角色：小鲨鱼像新手教练，聪明但亲切

## 交付物

- 竖屏成片：1080 x 1920, 60-90 秒
- 封面图：1080 x 1920
- 字幕文件：SRT
- 项目说明：README
- 脚本和分镜：Markdown

## 制作顺序

1. 定稿旁白：先控制在 180-230 个中文字左右。
2. 做分镜：把每句话对应到画面，不让画面只是跟着字动。
3. 生成或绘制素材：小鲨鱼角色、牌桌、扑克牌、筹码、信息图标。
4. 剪辑：先铺旁白，再对齐画面和字幕。
5. 声音：加轻快背景音乐，音量低于旁白。
6. 导出：检查字幕、封面、首 3 秒吸引力。
